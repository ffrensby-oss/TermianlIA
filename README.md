# Proyecto IA Terminal

Este proyecto es algo sencillo que cualquiera puede hacer. La idea es realizar una pregunta sencilla a la IA desde la misma terminal y obtener una respuesta veloz (estoy trabajando en la velocidad, jajaja).

## Instalación

Por comodidad, recomiendo realizar los siguientes pasos:
1. Configura la ruta al entorno virtual de este repositorio y la ruta del script, en el archivo `ia`.
2. Mueve el archivo `ia` a `/usr/local/bin/`.

De esta forma, podrás usarlo desde cualquier parte del sistema.

---

*Nota: Ojalá al que lea esto le salga un error inesperado en la distro.... y que tenga que reinstalar el sistema.*
